# 個人資金規劃與記帳：手機版部署說明

這是一個靜態網站，可以放到 GitHub Pages、Netlify 或任何可託管 HTML/CSS/JavaScript 的服務。

## 你會得到什麼

- 手機可開的記帳網址
- 可加入手機主畫面的圖示
- 支援基本離線快取
- 資料存在手機瀏覽器本機
- 可用「備份資料」匯出完整 JSON 備份

## 重要提醒

這個版本沒有雲端同步。手機和電腦各自會有自己的本機資料。

如果你主要用手機記帳，請固定在手機上使用同一個瀏覽器，並且每週或每月按「備份資料」。

## Netlify 部署

1. 前往 Netlify。
2. 選擇新增網站。
3. 使用拖放部署時，請上傳這個資料夾中的全部檔案。
4. 部署完成後，Netlify 會給你一個 HTTPS 網址。
5. 用手機打開該網址。
6. 加到手機主畫面。

## GitHub Pages 部署

1. 建立一個新的 GitHub repository。
2. 把這個資料夾中的全部檔案放到 repository 根目錄。
3. 到 repository 的 Settings。
4. 進入 Pages。
5. Source 選擇要發布的 branch。
6. 儲存後等待 GitHub Pages 產生 HTTPS 網址。
7. 用手機打開該網址。
8. 加到手機主畫面。

## iPhone 加到主畫面

1. 用 Safari 打開網站。
2. 點分享按鈕。
3. 選「加入主畫面」。
4. 名稱可改成「資金記帳」。
5. 點加入。

## Android 加到主畫面

1. 用 Chrome 打開網站。
2. 點右上角選單。
3. 選「加到主畫面」或「安裝應用程式」。
4. 名稱可改成「資金記帳」。
5. 點加入。

## 每週備份建議

1. 打開網站。
2. 按「備份資料」。
3. 把下載的 JSON 檔存到 iCloud Drive、Google Drive 或電腦。

## 換手機或換瀏覽器

1. 在舊手機按「備份資料」。
2. 把 JSON 備份檔傳到新手機。
3. 在新手機打開網站。
4. 按「匯入備份」。
5. 選擇 JSON 檔。
